## 第一种输入方式
>例如：    
1 2 3 4 .......  
1,2,3,4 .......  
>9999999999     //特别大的整数用字符串输入  
    
条件：以上针对 __未知输入长度__ 的数据和较大的整数输入。  
方法：可以使用字符串读入,然后分别使用`" "`、`"，"`和`""` 为分割并最终转为int数组。(文中使用空格分割) 

## 第二种输入方式
例如：  
>1，2，3 ....  
3，4，5 ....  
.... ...... ..........    
  
条件：针对未知大小的二维数组输入  
注意：一般题目会有字符串流结束标志(EOF)会自动结束输出    
&emsp; &emsp;&nbsp; 自己IDE调试需输入`CTRL+D`以中止输入 或者 输入俩次回车即可
