## 简介：
题目大多来自leetcode和互联网上公开的笔试题目。（目前是主要是c++）   
学习算法的一些想法和c++知识点 
 
## 提交说明:
* 欢迎各位fork补充，并一起学习打卡。  
* 你可以提交任何学习算法中的代码和想法，建议以（代码+文本）放置在一个文件夹里提交。
* 提交的最小单位是一个文件夹，文件的名字为可以是 __题目的名字+涉及的算法名称+您的名字__ 添加即可;   
  文件可按照分类放置在已有的项目文件夹中。 
* 如果您的文件不在上述分类可自己新建 文件层级及命名参考项目的文件
